Home

The goal of this site is to create a system that contains accurate and easily accessible information about the aLIGO channels. We are currently in an early development and exploration phase during which time this site may be subject to frequent changes and updates.

If you encounter any problems or have suggestions or useful observations, send email to: cis-support@gravity.phys.uwm.edu or submit a bug report or feature request on bugs.ligo.org.

Searching / Filtering

Channel queries are based on case-insensitive sub-string matching. Spaces indicate AND and the '|' character indicates OR. AND takes precedence. For example, "PSL TEMP | sus- laser" will retrieve all channels with PSL and TEMP in their names along with all channels with SUS- and LASER in their names.

Editing

If you have editing priviledges, the text [Edit] will appear next to editable entries. It doesn't look like a link, but if clicked on, an edit form will appear. The form will ask for a short description and a long description. The short description should be a one liner. The long description can be any HTML text, so you can enter links, images, whatever you require. Yes, it's a bit risky allowing any HTML, but we trust you. Both fields are optional, so you can enter one or the other or both, though it is best to at least have a short description.

Entering a description for a term defines that term for all channels. So if it means something different in different channels, one would want to enumerate all possible meanings across channels. (but don't worry about doing that up front if you don't know whether there are other meanings or not -- presumably others can correct omissions)

It has been reported that sometimes submitting an edit fails, as if a connection has timed out. Reloading the page and resubmitting fixes the problem, but that fails to remember your edits, so if this happens to you, you might save your entered text before reloading. This is a bug that is being investigated.
